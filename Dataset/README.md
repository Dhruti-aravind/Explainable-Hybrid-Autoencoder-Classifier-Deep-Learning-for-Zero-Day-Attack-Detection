This project utilizes the UNSW-NB15 dataset, a comprehensive and realistic network intrusion dataset containing normal traffic and nine diverse attack categories. It includes modern synthetic and real network flows with rich feature representations, making it well-suited for training and evaluating deep learning models. The dataset enables effective detection of both known and zero-day cyber attacks in intrusion detection systems.

Link: [https://www.kaggle.com/datasets/dhoogla/unswnb15](https://www.kaggle.com/datasets/dhoogla/unswnb15)
